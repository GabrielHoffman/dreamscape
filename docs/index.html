Dreamscape ecosystem

Recent advances in single cell/nucleus transcriptomic technology have enabled collection of population-scale datasets to study cell type specific gene expression differences associated with disease state, stimulus, and genetic regulation. The scale of these data, complex study designs, and low read count per cell mean that characterizing cell type specific molecular mechanisms requires a purpose-built analytical framework.

Dreamscape is a set of R / Bioconductor packages that powers differential expression analysis, variance decomposition, cell-type composition analysis, gene set enrichment, and genetic regulation of gene expression on population-scale single cell datasets.
lucida: Scalable differential expression analysis at the single cell level1
GenomicDataStream: A scalable interface between genomic data and analysis underneath R2
fastglmm: Efficiently Fit Generalized Linear Mixed Model with a Single Random Effect on Massive Datasets3
BatchRegression: Fit regression models on large-scale datasets
dreamlet: Scalable differential expression analysis of single cell transcriptomics datasets with complex study designs 4
crumblr: Count ratio uncertainty modeling based linear regression 5
dream: Powerful differential expression analysis for repeated measures designs 6
variancePartition: Interpreting drivers of variation in complex gene expression studies with linear mixed models 7
